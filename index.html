scViewer: scRNA-seq Made Easy

A Scanpy-based single-cell analysis application designed for researchers who prefer not to code. Features include the complete single-cell analysis pipeline, h5ad support, cell type annotation, differential expression analysis, and customizable visualizations.

ArcCell: scRNA-seq Annotation Tool

A metric-learning model using ArcFace architecture with SHAP-based gene explanations. Benchmarked on single-cell datasets of mouse gonadal tissue for improved cell type annotation accuracy.

PyTorch SHAP ML

Current Positions:
